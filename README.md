## Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina **WEB**, ministrada pelo professor **Vitor Pinheiro**, no curso de **Redes de Computadores** da **Universidade Federal do Ceará** (campus Quixadá).  

O sistema implementa uma interface moderna e interativa para exibição e gerenciamento de posts, com recursos de listagem, busca, filtros e paginação, proporcionando uma experiência fluida ao usuário.  

---

## Funcionalidades

**Listagem de Posts** — Exibe uma tabela com todos os registros.  
**Busca por Nome ou E-mail** — Permite encontrar rapidamente um post específico.  
**Filtro por Cidade** — Mostra apenas os itens de uma localidade escolhida.  
**Página de Detalhes** — Exibe informações detalhadas sobre um item selecionado.  
**Página “Sobre”** — Apresenta informações gerais sobre o projeto.  
**Paginação Automática** — Mostra 5 itens por página, com navegação fácil.  
**Filtros Salvos no LocalStorage** — Mantém as preferências do usuário mesmo após atualizar a página.  
**Modal de Posts** — Exibe o conteúdo de forma destacada sem sair da página.  
**Loading Animado (Spinner)** — Indica o carregamento de dados de forma elegante.  
**Contagem de Posts** — Exibe o número total de registros na tabela principal.
